# Plugin GAS Framework

## 개요

GAS를 기반으로 멀티플레이 게임 제작을 위한 프레임워크입니다.

## 목표 게임 장르

- FPS / TPS
- RPG
- 크래프팅

## 현재 진행중인 작업

- 장비창
- 총기
- 1인칭 / 3인칭 애니메이션

## 작업 예정

- 메시징 서브 시스템을 활용한 UI 바인딩
- 캐릭터 선택
- 파쿠르

## 완료된 작업

- AttributeSet 작성 편의를 위한 매크로 및 메서드
- 향상된 입력 액션과 어빌리티 입력 바인딩
- 캐릭터 움직임에 따라 대응하는 게임플레이 태그 자동 적용
- 캐릭터 상태 머신
- 게임플레이 태그와 애님 인스턴스 부울 변수 바인딩
